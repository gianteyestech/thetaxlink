# The Tax Link Website

Modern React + Vite website exported from Base44 and cleaned for independent GitHub deployment.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Base44 SDK and Base44 Vite plugin have been removed.
- Routing is handled by React Router.
- UI, layout, animations, and existing image URLs are preserved.
